# Undangan Khitan Digital

Website statis siap deploy ke GitHub Pages.

## Cara pakai
1. Upload semua file ke repository GitHub.
2. Pastikan `index.html`, `style.css`, `script.js`, dan folder `assets` berada di root repository.
3. Aktifkan GitHub Pages: Settings > Pages > Deploy from a branch > main > /root.

## Data yang perlu diganti
- Nomor WhatsApp RSVP di `script.js` bagian `phone`.
- Alamat dan link Google Maps di `index.html`.
- Rekening di `index.html`.
- Nama orang tua di section penutup.
